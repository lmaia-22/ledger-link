Implement the COM bridge as two files: `bridge-client.ts` (runs in main Fastify process) and `bridge-host.ts` (runs in forked child process).

**src/sage-bridge/bridge-client.ts:**
- Import `fork` from `node:child_process`, `crypto` from `node:crypto`
- Import `BridgeRequest`, `BridgeResponse` from `./types.js`
- Maintain module-level `child: ChildProcess | null` and `pending: Map<string, { resolve, reject, timer }>`
- `ensureBridge()`: If child is null or exited, fork `bridge-host.ts` (use `new URL('./bridge-host.js', import.meta.url).pathname` for path resolution). Register `child.on('message', handler)` that resolves/rejects the matching pending Promise by `msg.id`. Register `child.on('exit', handler)` that rejects all pending with "Bridge process crashed".
- `callBridge<T>(req: Omit<BridgeRequest, 'id'>, timeoutMs = 30000): Promise<T>`: Generate UUID id, create Promise, store in pending map, set a setTimeout that rejects with "Bridge timeout" and removes from pending after `timeoutMs`. Call `ensureBridge().send({ ...req, id })`. Return the Promise.
- `shutdownBridge(): Promise<void>`: Send a `{ type: 'SHUTDOWN' }` message (no id needed), kill child after 5s if still alive, clear pending map.
- Export `ensureBridge`, `callBridge`, `shutdownBridge`.

**src/sage-bridge/bridge-host.ts:**

**src/server.ts** — App factory:
```typescript
import Fastify from 'fastify';
import staticPlugin from '@fastify/static';
import path from 'node:path';
import { companiesRoutes } from './api/routes/companies.js';
import { transactionsRoutes } from './api/routes/transactions.js';

export async function createApp(opts?: { serve_static?: boolean }) {
  const app = Fastify({ logger: true });

  // API routes (registered first — matched before static wildcard)
  await app.register(companiesRoutes, { prefix: '/api' });
  await app.register(transactionsRoutes, { prefix: '/api' });

  // Serve Vite build output in production
  if (opts?.serve_static !== false) {
    const distPath = path.join(import.meta.dirname, '../../client/dist');
    try {
      await app.register(staticPlugin, {
        root: distPath,
        wildcard: false,
      });
      // SPA catch-all — sends index.html for all unmatched routes
      app.setNotFoundHandler((_req, reply) => {
        reply.sendFile('index.html');
      });
    } catch {
      // client/dist may not exist in dev mode — that's OK
      app.log.warn('client/dist not found, skipping static file serving');
    }
  }

  return app;
}
```

**src/index.ts** — Entry point:
```typescript
import { createApp } from './server.js';
import { shutdownBridge } from './sage-bridge/bridge-client.js';

const port = Number(process.env.PORT ?? 3000);
const host = process.env.HOST ?? '0.0.0.0';

const app = await createApp();

// Graceful shutdown
const shutdown = async () => {
  app.log.info('Shutting down...');
  await shutdownBridge();
  await app.close();
  process.exit(0);
};
process.on('SIGINT', shutdown);
process.on('SIGTERM', shutdown);

await app.listen({ port, host });
app.log.info(`Ledger Link server listening on http://${host}:${port}`);
```
